# Theme Color Specifications

## Color Contrast Compliance

All color combinations in this portfolio meet or exceed WCAG 2.1 AA standards (4.5:1 for normal text, 3:1 for large text).

## Implementation Notes

### Dynamic Theme Switching

The theme system uses CSS custom properties that are dynamically updated via JavaScript when the user switches themes. This allows for instant theme changes without page reload.

### Persistence

Theme preferences are stored in localStorage:
- `portfolio-theme`: Stores light/dark mode preference
- `advanced-theme`: Stores specific theme variant (ocean, sage, midnight, slate)

### Auto-Switching

When toggling between light and dark modes, the system automatically selects an appropriate theme variant:
- Light mode → Ocean (default) or previously selected light theme
- Dark mode → Midnight (default) or previously selected dark theme

### Color Blindness Considerations

All themes avoid relying solely on color to convey information:
- Interactive elements have hover states with visual feedback
- Focus indicators provide clear outlines
- Icons supplement color-coded information
- Text labels accompany status colors
